Please note that by default the advanced settings are configured to the defaults that work with v2 of Sonarr and v0 of Radarr.

If you are using the pre-release v3.0 of either, then please head to the options page, advanced tab and setup the v3.0 defaults.

If you have issues getting these settings working then the pre-release code may have changed, please raise an issue on the project GitHub repo.
